Technology
How detection works
01
Face Detection
OpenCV's Haar Cascade localizes facial regions in the image before classification, isolating each face patch independently.
02
CNN Classification
Each face patch is fed into a convolutional neural network trained on 140K+ labeled images from the Kaggle dataset.
03
Confidence Score
Softmax output is averaged across all detected faces, producing a final authenticity score with a full human-readable explanation.
